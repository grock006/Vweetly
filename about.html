Vweetly Helps Organize Tweets From Your Favorite TV Shows!


Vweetly is my final project for the Front End Web Development class at General Assembly Los Angeles. It was created with HTML, CSS and Javascript. Vweetly uses Twitter's API and "Twitterfetcher" javascript (developed by Jason Mayes) to import Twitter timeline ID's into pre-selected categories: people, recaps and #trending topics relating to a selected television show.
